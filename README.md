# Municipes

Controle de cadastro de Municipes com endereço.

## Descrição

Cadastro para Municipes com registro de endereço, busca por CEP e inserção de imagem.

## Passos iniciais

### Dependências

* Docker

### Instalação

* Docker
* docker-compose

### Executando aplicação

```
./start.sh
```

### Executando testes

```
./rspec.sh
```
